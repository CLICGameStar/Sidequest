Règles générales

  • Les rencontres sont gérées par les joueur·euse·s.
  • À la fin de chaque rencontre, veuillez reporter l'issue de la partie sur le formulaire ci-dessous.
  • En cas de litige, si aucune preuve n'est apportée (e.g., une capture d'écran), les deux joueur·euse·s seront éliminé·e·s.
  • En cas de non-respect du délai imposé pour un match, le vainqueur sera désigné par tirage au sort afin de ne pas pénaliser les autres participant·e·s.

Terminologie

  • Un "jeu" est défini comme l'instance de combat dont l'issue est déterminée par la perte de toutes les vies d'un·e joueur·euse.
  • Un "set" est défini comme le meilleur des trois (Best-of-3) ou le meilleur des cinq jeux (Best-of-5) qui est joué jusqu'à ce qu'un·e joueur·euse ait gagné la majorité du set. (Les sets multiples ne sont joués que dans le cas où le·la joueur·euse de l'arbre inférieur remporte un set lors de la grande finale).
  • Un "match" est défini comme le set ou l'ensemble de sets dans lequel un·e joueur·euse a officiellement perdu et le·la joueur·euse gagnant·e se qualifie pour le tour suivant du tournoi.

Règles spécifiques au jeu

  • Les matches se jouent en 1 contre 1, sur des cartes aléatoires parmi le type autorisé (voir paramètres ci-dessous).
  • La finale des vainqueurs, la finale des perdants et la grande finale se jouent en 3 jeux gagnants (Best-of-5) ; Tous les autres matches se jouent en 2 jeux gagnants (Best-of-3).
  • Toutes les légendes sont autorisées, et les joueur·euse·s peuvent changer de légende entre chaque jeu.

Paramètres de jeu

Chaque partie doit être créée avec les règles de jeu suivantes :

  • Mode de jeu : Stocks
  • Équipes : Désactivé
  • Vies : 3
  • Temps de jeu : 8:00
  • Dégats : 100%
  • Weapons & Gadgets : No Gadgets
  • Fonctionnalités à tester : Désactivé
  • Cartes à tester : Désactivé
  • Type de cartes : 1v1
  • Nbr joueurs max. : 2

Foire Aux Questions

Générales

Puis-je obtenir une extension de délai pour mon match ?

Il est préférable de respecter les délais, mais s'il vous est impossible de disputer votre rencontre vous pouvez contacter @LaClimSx sur Telegram.

Comment faire si mon adversaire conteste ma victoire ?

Comme mentionné dans les règles, en cas de conflit, si aucun·e joueur·euse ne peut apporter de preuve de sa version, les deux joueur·euse·s seront éliminé·e·s. Nous vous conseillons donc de prendre une capture d'écran du résultat pour prévenir ce problème.
